# Анкета — задание по JavaScript (формы, cookie, JSON, Local Storage)

## Что сделано

- **HTML‑форма** по образцу с полями: `Имя`, `Фамилия`, `E‑mail`, `Комментарий`, кнопки `OK` и `Отмена`.
- **Вывод значений формы** на текущую HTML‑страницу после отправки.
- **Запись данных в cookie** и вывод значений из cookie при перезагрузке страницы.
- **Валидация полей**:
  - HTML5‑атрибуты (`required`, `type="email"`, `minlength`, `maxlength`).
  - Проверка на пустые поля в JavaScript.
  - Проверка формата E‑mail через регулярное выражение.
- **JSON‑представление**: данные формы сериализуются в JSON, показываются на странице и дополнительно сохраняются в `localStorage`.

## Структура

- `index.html` — страница с формой и блоками для вывода результатов.
- `style.css` — оформление формы и результатов.
- `script.js` — логика работы с формой, cookie, валидацией и JSON.

## Как запускать в OSPanel

1. Положите папку проекта в каталог домена OSPanel (уже находится в `d:\OSPanel\home\карточка`).
2. В настройках OSPanel убедитесь, что создан домен, указывающий на эту папку (например, `http://карточка/`).
3. Откройте домен в браузере и протестируйте работу формы.
